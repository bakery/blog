# The Blog

## Setup 
**Jekyll 2.x required**

```
gem install bundler
bundle install
```

## New post

```
octopress new post "Post Title"
```

To enable sharing and comments for the new post edit the header in the .md file and add this

```
share: true
comments: true
```
